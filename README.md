# sipmle-nodejs-app
This repository been create for 'Starter Task'

## Installation
```sh
$ npm install -g mocha
$ npm install
```
## Launching
```sh
$ export NODE_ENV=development
$ npm start
$ npm test
```

## Example routes

Create checkin
> host:port/checkin?username=Anton&location=Taganrog

Search by location
> host:port/query?location=Taganrog

Search by username
> host:port/query?username=Anton
